# Penjelasan Class Diagram

Class diagram ini menggambarkan struktur data sistem inventaris Mixue yang terdiri dari UserModel untuk data pengguna, BarangModel untuk master barang, BarangMasukModel/BarangTerpakaiModel/BarangExpiredModel untuk transaksi stok, serta OrderModel dengan OrderItemModel untuk penjualan dan RefundModel untuk pengembalian.
